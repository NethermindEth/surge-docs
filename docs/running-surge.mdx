# Getting Started

This guide will help you set up and run Surge rollup.

## Prerequisites

- Intel processor with SGX (Software Guard Extensions) support for the prover
- Docker and Docker Compose
- Git
- 16GB RAM minimum
- 50GB free disk space

## Steps

1. Install Kurtosis ([installation guide](https://docs.kurtosis.com/quickstart))
2. Deploy L1 ([guide](/Guides/running-surge/deploy-l1.mdx))
3. Deploy L2 ([guide](/Guides/running-surge/deploy-l2.mdx))
4. Set up Provers ([guide](/Guides/running-surge/provers/index.mdx))
5. Deploy Relayer ([guide](/Guides/running-surge/deploy-relayer.mdx))
6. Launch Bridge UI ([guide](/Guides/running-surge/deploy-bridge-ui.mdx))

Error Codes

Error codes from the surge-alethia-real-time-driver branch of surge-taiko-mono. Core real-time proving errors are listed first.

Core: RealTimeInbox.sol (layer1/core/impl/)​

ErrorCodeDescription
AlreadyActivated()—Inbox already has a genesis block hash
MaxAnchorBlockTooOld()—Referenced L1 block is older than 256 blocks
InvalidGenesisHash()—Zero hash passed to activate()
NotActivated()—propose() called before activate()
SignalSlotNotSent(bytes32 slot)—Signal slot doesn't exist on L1 SignalService

Core: SurgeVerifier.sol (layer1/surge/)​

ErrorDescription
Surge_CallerIsNotInbox()Only the inbox can call verifyProof
Surge_InvalidProofBitFlag()Invalid proof type flag in sub-proof
Surge_NumProofsThresholdNotMet()Not enough distinct proofs submitted
Surge_AlreadyMarkedForProposalId()Proof type already submitted for this proposal
Surge_InstantUpgradeNotAllowed()Verifier upgrade timing violation

Core: ZiskVerifier.sol (layer1/verifiers/)​

ErrorDescription
ZISK_INVALID_PROGRAM_VKEY()Program verification key not trusted
ZISK_INVALID_CHAIN_ID()L2 chain ID invalid
ZISK_INVALID_REMOTE_VERIFIER()Remote verifier address is zero
ZISK_INVALID_PARAMS()Proof too short
ZISK_INVALID_PROOF()SNARK proof verification failed

Core: Anchor.sol (layer2/core/)​

ErrorDescription
AncestorsHashMismatch()L1 ancestor hash chain validation failed
InvalidBlockNumber()Anchor block number invalid
InvalidL1ChainId()Mismatched L1 chain ID
InvalidL2ChainId()Mismatched L2 chain ID
InvalidSender()Caller isn't GOLDEN_TOUCH_ADDRESS
InvalidAddress()Zero address passed

Shared: SignalService.sol​

ErrorDescription
SS_CHECKPOINT_NOT_FOUND()Checkpoint not found in store
SS_EMPTY_PROOF()Empty merkle proof provided
SS_INVALID_BLOCK_ID()Invalid block ID in proof
SS_INVALID_CHECKPOINT()Invalid checkpoint structure
SS_INVALID_PROOF_LENGTH()Merkle proof length mismatch
SS_SIGNAL_NOT_RECEIVED()Signal slot not marked as received
SS_UNAUTHORIZED()Caller unauthorized
